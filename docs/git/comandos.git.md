# Comando de GIT (Iniciales)
## Requerimientos
- Descargar GIT (https://git-scm.com/)
- Crear una cuenta de GITHUB (https://github.com/)
## Configurar GIt
- Ejecutar los siguientec comandos
```
git config --global user.name "Su Nombre"
git config --global user.email "su.correo@gmail.com"
```
